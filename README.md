# MemeSniper Agent

**AI-powered Trading Agent that watches new Solana memecoin launches, scores them with Qwen, and signals you before retail piles in.**

Built for the **Bitget AI x Crypto Trading Hackathon 2026**.

---

## What It Does

MemeSniper Agent is an end-to-end AI trading pipeline:

1. **Watches** new Solana token launches in real time
2. **Scores** each token 0-100 using **Qwen LLM** (Alibaba Cloud's flagship model)
3. **Classifies** tokens as **BUY / WATCH / SKIP** based on liquidity, holder count, and token age
4. **Displays** results in a live Streamlit dashboard
5. **Persists** all verdicts to JSON for backtesting

The agent answers one question: *Is this new memecoin worth buying, or is it a scam/rugpull?*

## Quick Start

### 1. Install dependencies

\`\`\`bash
pip install -r requirements.txt
\`\`\`

### 2. Add your API keys

Copy `.env.example` to `.env` and fill in:

\`\`\`bash
QWEN_API_KEY=your_qwen_key_here
BITGET_API_KEY=your_bitget_key_here  # Optional
BITGET_API_SECRET=your_bitget_secret_here  # Optional
\`\`\`

### 3. Run the agent

\`\`\`bash
python -m src.agent
\`\`\`

Output:
\`\`\`
   =========================================================
     M E M E   S N I P E R   A G E N T
     AI-Powered Solana Memecoin Hunter
   =========================================================

[INFO] Cycle 1/5
[INFO]   Myro18 -> 87 (BUY)
[INFO]   Doge244 -> 44 (SKIP)
[INFO]   Pepe216 -> 89 (BUY)
...

Done. 25 verdicts saved to dashboard.
\`\`\`

### 4. Launch the dashboard

In a separate terminal:

\`\`\`bash
python -m streamlit run dashboard/app.py
\`\`\`

Open http://localhost:8501 in your browser. Click **"Run Agent Now"** to refresh data.

## How Scoring Works

Each token is scored on three dimensions:

| Factor | Weight | Why |
|--------|--------|-----|
| **Liquidity (USD)** | 0-50 | Higher liquidity = harder to rug, more credible |
| **Holder count** | 0-30 | More holders = distributed risk, real demand |
| **Token age** | 0-20 | Newer launches with strong metrics are interesting |

**Verdict thresholds:**

- **0-30: SKIP** - Almost certainly a scam or rugpull
- **31-60: WATCH** - Mixed signals, monitor closely
- **61-80: BUY** - Solid metrics, likely good entry
- **81-100: STRONG BUY** - High conviction signal

The LLM (Qwen) receives the same data plus context about memecoin trading patterns, then returns a refined score with natural-language reasoning.
